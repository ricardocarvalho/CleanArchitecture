# CleanArchitecture
Principais conceitos relacionados com a Clean Architecture.

bBas práticas que aplicadas permitem criar aplicações ASP .NET Core com uma arquitetura limpa.  Os fundamentos apresentados baseiam-se na Clean Architecture que refere-se à organização do projeto de forma que seja fácil de entender e fácil de mudar conforme o projeto cresce. Isso não acontece por acaso. É preciso um planejamento intencional para que isso ocorra.
